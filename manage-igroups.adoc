= Manage the igroups for a file system in NetApp Workload Factory
:icons: font
:imagesdir: ./media/

[.lead]
Use the NetApp Workload Factory console to manage igroups and control client access for FSx for ONTAP block devices. You can view igroup details, manage client access, and delete igroups.

.Before you begin
* You must associate a link to manage igroups. link:https://docs.netapp.com/us-en/workload-fsx-ontap/create-link.html[Learn how to associate an existing link or to create and associate a new link]. After you associate the link, return to this operation.
* You must have an existing igroup to view and manage.

== Manage client access for an igroup
You can manage client access for an existing igroup at any time.
 
.Steps
. Log in using one of the link:https://docs.netapp.com/us-en/workload-setup-admin/console-experiences.html[console experiences^].
. Select the menu image:workloads-menu-icon.png[The hamburger menu icon is used to navigate to workloads like storage, eda, ai, databases, vmware, and administration.] and then select *Storage*.
. From the Storage menu, select *FSx for ONTAP*.
. From *FSx for ONTAP*, select the *Block devices* tab. 
. Select the resource type *Initiator groups (igroups)* to view existing igroups.
. Go to the actions menu for the block device and select *Manage client access*.
. Review the client access details displayed for the igroup.
. To make changes to client access, select *Edit client access*.
. In the *Edit client access* dialog, you can edit the following:

* *igroup name*
* *igroup description*
* *Storage VM name*
* *Block device name*
* *Operating system type*
* *Host initiators*
. Select *Apply*.

== Delete an igroup 
You can delete an existing igroup when it is no longer needed.    

.Steps
. Log in using one of the link:https://docs.netapp.com/us-en/workload-setup-admin/console-experiences.html[console experiences^].
. Select the menu image:workloads-menu-icon.png[The hamburger menu icon is used to navigate to workloads like storage, eda, ai, databases, vmware, and administration.] and then select *Storage*.
. From the Storage menu, select *FSx for ONTAP*.
. From *FSx for ONTAP*, select the *Block devices* tab. 
. Select the resource type *Initiator groups (igroups)* to view existing igroups.
. Navigate to the actions menu for the block device and then select *Delete initiator group*.
. In the Delete initiator group (igroup) dialog, type "delete" to confirm that you want to delete the igroup, and then select *Delete*.
